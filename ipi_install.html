Obtaining and installing i-PI

You can download a snapshot of the first release of i-PI, that also contains a manual explaining installation and usage. It is however recommended that you fetch the most recent update from i-PI's public github repository. The development of the next version of i-PI is ongoing in a private git repository. If you want to contribute, or check out the latest developments you can e-mail me.

Obtaining and installing a force client

i-PI is distributed with a simple driver code that can be used to run the examples, and perform some simple calculations. Clearly, it is most useful when used together with an electronic structure code, or a code that implements complex and computationally-demanding empirical potentials. The distribution contain some patches for several electronic structure codes, and it should be simple to modify another program taking inspiration from the driver and the existing patches.

A few programs offer already an i-PI interface in the latest/development version, including CP2K, FHI-AIMS, DFTB+ and LAMMPS. Again, please e-mail me if you are interested in having an i-PI interface in a code you are using or developing.
