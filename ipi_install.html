Obtaining and installing i-PI

You can download a snapshot of the first release of i-PI, that also contains a manual explaining installation and usage. It is however recommended that you fetch the most recent update from i-PI's public github repository. The development of the next version of i-PI is ongoing in a private git repository. If you want to contribute, or check out the latest developments you can e-mail me.
